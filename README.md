Bouncing Text Applet (Java)
A simple Java Applet that displays a name moving horizontally across the screen using AWT and multithreading.
Features
Smooth left-to-right text animation
Customizable text and speed
Black background with green text
Uses a separate thread (Runnable) for animation
Files
BouncingTextApplet.java – Handles the animated bouncing text
Main.java – Simple console-based Java entry point
Note
Java Applets are deprecated in modern browsers and are mainly used for learning and practice purposes.
